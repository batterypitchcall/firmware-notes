# Battery PitchCall — firmware update notes

Public metadata the **coach** fetches during a firmware-update check to decide
whether a newer build exists and to show *"here's what's changed"* before
downloading. The firmware **binary** itself is delivered privately via Memfault;
this repo only holds the version + release-note text for display.

- `coach-fw/latest.json` — `{ software_type, version, notes }` for the coach.

Generated by `tools/ota/gen_latest_json.py` in the coach-controller repo as part
of cutting a release, so the advertised version always matches the Memfault release.
